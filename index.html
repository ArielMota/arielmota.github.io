  • API RESTful Gestão de Eventos

    Desenvolvi uma API RESTful com Django e Django REST Framework (DRF) para a gestão de eventos e participantes, garantindo segurança, escalabilidade e alta performance. A API utiliza autenticação JWT com SimpleJWT, proporcionando um sistema seguro e eficiente de gerenciamento de acessos. Para otimizar a performance e experiência do usuário, implementei paginação com LimitOffsetPagination, permitindo consultas rápidas e controle eficiente do volume de dados. Além disso, a API conta com um fluxo de deploy automatizado, reduzindo erros manuais e garantindo atualizações contínuas e confiáveis, facilitando a manutenção e escalabilidade do sistema.
